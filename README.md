# Pandas Data Manipulation & Analysis

This repository contains a collection of Python scripts demonstrating data processing, cleaning, and analysis using the **Pandas** library. These projects form the data-handling foundation for my work in **Data Science & AI/ML**.

## 📊 Core Functionalities Explored

* **Data Ingestion:** Loading and inspecting datasets from CSV, Excel, and JSON formats.
* **Data Cleaning:** Handling missing values (`NaN`), deduplication, and data type normalization.
* **Wrangling & Transformation:** Advanced indexing with `.loc` and `.iloc`, and applying complex transformations using `apply()` and `map()`.
* **Aggregation:** Using `groupby` and pivot tables to extract meaningful summaries from raw data.

## 📁 Repository Structure

* `1. DataFrame Basics.py`: Introduction to Series and DataFrames, including basic attributes and methods.
* `2. Data Cleaning.py`: Practical examples of cleaning "messy" data—essential for reliable AI models.
* `3. Analysis & Grouping.py`: Using aggregation to find trends within categorical data.
* `4. Integration.py`: Preparing structured dataframes for visualization in **Matplotlib**.

The scripts in this repository directly support my larger projects:
